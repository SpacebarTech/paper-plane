# Paper Plane

The official component library for JETS Technologies, LLC.
This project conforms to our internal styleguide and design
language.

## Installation

### `npm`

```bash
npm i @jetstech/paper-plane
```

### `yarn`

```bash
yarn add @jetstech/paper-plane
```
